# {{Project name}} — Site Inventory

One block per page (a.k.a. **page inventory** in IA practice). Group by section (public, account, by-role areas).

**Status values:** `Live` (in production) · `Planned` (stub or partial) · `DELETE` (flagged for removal) · `Deprecated`

**Field reference:**
- **Purpose** — one-line reason the page exists
- **Audience** — which role(s) reach it
- **Content (current)** — what's actually rendered today
- **Actions** — what the user can do, including cross-section navigation
- **Status** — lifecycle state
- **Notes** — TODOs, open questions, deletion flags
