# {{Project name}} — Architecture Decisions

ADR (Architecture Decision Record) log. Each entry records a decision: what was chosen, why, and what alternatives were ruled out. When you find yourself re-asking a question, search here first.

**Field reference:**
- **Date** — ISO `YYYY-MM-DD` when the decision crystallized
- **Status** — `Accepted` | `Superseded by ADR-N` | `Reverted`
- **Context** — problem (1-2 sentences)
- **Decision** — what was chosen (1 sentence)
- **Reasoning** — why (2-4 sentences)
- **Alternatives ruled out** — what was considered and why each was rejected (1-2 bullets)
- **Consequences** — what this implies going forward (1-2 bullets)
- **Source** — where the canonical statement lives (CLAUDE.md section, memory file, plan file, this ADR if it's the source)
